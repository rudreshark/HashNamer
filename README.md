## Usage

Run the script:
```bash
python3 hashnamer.py
```

You will see a banner and then the interactive prompt:
```
INPUT >
```

Paste any encoded string, hash, ciphertext, or hex dump and press Enter.

### How it works (fully automatic)
1. **Encoding detection** ‚Äď The tool scans the input against dozens of regular expressions.
2. **Auto‚ÄĎchaining** ‚Äď It decodes the input, then re‚ÄĎscans the result, repeating until no more known patterns are found.
3. **Hash identification** ‚Äď If the input (or any decoded step) matches a hash signature, it is flagged.
4. **Cracking** ‚Äď For crackable hashes (MD5, SHA1, NTLM, etc.), the tool instantly:
   - Tries the built‚ÄĎin wordlist.
   - If not found, scans any system wordlist it finds (rockyou.txt, etc.).
   - Reports the plaintext password and automatically checks it against HIBP.
5. **Final output** ‚Äď The last decodable / cracked value is shown.

## HIBP (Have I Been Pwned) Integration

When a password is cracked, the tool automatically queries the **Have I Been Pwned** API using the k‚ÄĎanonymity model:
- It sends only the first 5 characters of the SHA‚ÄĎ1 hash.
- The API returns a list of matching hash suffixes and their breach counts.
- Your password **never leaves your machine** in plain text.

If the password is found, you'll see something like:
```

```

## Wordlist Management

The tool automatically scans common system paths for wordlists:
- `/usr/share/wordlists`
- `/usr/share/dict`
- `/usr/share/seclists`
- `/opt/wordlists`

It prioritises `rockyou.txt` (uncompressed) if found. The built‚ÄĎin list (~130 entries) is always tried first and is enough for most CTF passwords.
